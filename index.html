ShExcala - Shape Expressions in Scala

Introduction

ShExcala is an implementation of Shape Expressions language that can be used to define the topology of RDF graphs.

ShExcala can be used as a standalone command line program and as a jar library that can be invoked from Scala or Java programs.

It has been used as the basis of the RDFShape online validator tool

You can see more information about how to use ShExcala in this link.
