ShExcala - Shape Expressions Implementation

Introduction

ShExcala is an implementation of the Shape Expressions language that can describe and validate the topology of RDF graphs.

ShExcala can be used as a standalone command line program and as a jar library that can be invoked from Scala or Java programs.

It has been used as the basis of the RDFShape online validator tool

You can see more information about how to use ShExcala in this link.
  • Packaged binaries (command line tool): shexcala-0.2.0.zip, shexcala-0.2.0.tgz
  • Latest release (binary jar): Jars and Javadocs. These jars can be used with Maven or SBT (see instructions) but not as a standalone jars because they need the Scala library in the classpath.
  • Source code: Github project
